# Vidly Django

A simple Django practice project for managing movie data.

This is a learning project and may evolve over time as new concepts are practiced.

## Features
- Movie list page
- Movie detail page
- Admin panel integration
- Basic API experimentation

## Tech Stack
- Python
- Django
- SQLite

## Purpose
This repository contains a practice project built while learning Django fundamentals such as:
- URL routing
- Views
- Templates
- Models
- Admin configuration
- Basic API integration

## Run locally
```bash
pip install pipenv
py -m pipenv install
py -m pipenv shell
python manage.py runserver
